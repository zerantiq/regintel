# Code of Conduct

This project follows a simple standard: be respectful, direct, and constructive.

## Expected Behavior

- Assume good intent and discuss the work, not the person.
- Give specific, evidence-based feedback.
- Welcome questions, corrections, and well-scoped disagreement.
- Keep issue and pull request discussions focused on the repository.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks
- Discriminatory or abusive language
- Repeated bad-faith disruption of issues or pull requests
- Publishing private or sensitive information without consent

## Enforcement

Project maintainers may remove comments, close discussions, or restrict participation when behavior harms the project or its contributors.

## Reporting

If conduct in this repository crosses the line, contact the maintainers through GitHub with links, screenshots, and relevant context.
